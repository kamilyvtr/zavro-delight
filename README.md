##  Sobre

**Zavro Delícia** é uma página de vendas de doces artesanais desenvolvida com HTML, CSS e JavaScript. O projeto entrega uma experiência de compra simples, responsiva.

---

## Funcionalidades

-  **Catálogo de doces** — exibição dos produtos com nome, foto e preço
-  **Carrinho de compras** — adição, remoção e contagem de itens
- **Design responsivo** — layout adaptado para mobile, tablet e desktop

##  Como Executar Localmente

```bash
# 1. Clone o repositório
git clone https://github.com/seu-usuario/zavro-delicia.git

# 2. Acesse a pasta do projeto
cd zavro-delicia

# 3. Abra no navegador
# direto pelo explorador de arquivos:
#   clique duas vezes no arquivo index.html
#
#  via extensão Live Server no VS Code:
#   botão direito no index.html → "Open with Live Server"
```
